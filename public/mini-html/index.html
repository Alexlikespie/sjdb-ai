Mini — trained in the stratosphere.

Mini is a fine-tuned, from-scratch language model trained by the SJDB Robotics Club. Pro was the teacher model that helped train it — the cloud endpoint you already know is the bigger sibling. Run Mini locally, fully on your own machine.

01 What it is

A small, hand-rolled language model. Q4_K_M quantised for everyday laptops. The teacher model — Pro — guided Mini's training, so it speaks with a similar voice at a fraction of the size.

02 Self-hosted

Mini is a self-hosted model and requires technical setup. You'll need LM Studio (free, runs on macOS / Windows / Linux) and a few minutes to point it at our preset.

03 Privacy

Once it's running, nothing leaves your computer. No telemetry, no server logs, no API keys. Great for coursework, sensitive prompts, and experimenting without rate limits.

Run it yourself

We recommend lmstudio.ai. Download the preset below, drop it into LM Studio, and you're a click away from chatting.
